gRPC in Node.js
===========================

PREREQUISITES
-------------

- `node`: This requires Node 0.12.x or greater.

INSTALL
-------

   ```sh
   $ cd node
   $ npm install
   ```

TRY IT!
-------

 - Run the server

   ```sh
   $ # from this directory
   $ node ./dynamic_codegen/greeter_server.js &
   ```

 - Run the client in another terminal

   ```sh
   $ # from this directory
   $ node ./dynamic_codegen/greeter_client.js
   ```
